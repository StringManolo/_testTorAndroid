# Check Tor

Dummy project running Tor bins cross compiled from [TB4A (Tor Bins For Android)](https://github.com/stringmanolo/tb4a/releases) into a webview (Browser).

![Screenshoot of the app running tor and the webview loading check.torproject.org](https://raw.githubusercontent.com/StringManolo/_testTorAndroid/refs/heads/main/fastlane/metadata/android/en-US/images/phoneScreenshots/1.jpg)

This project is for debug the Tor bins. Is also an example of how to run any ELF on Android.  
  
And as an example of how to run your own Tor compiled bins on your Android apk without using third party build like Orbot, TorBrowser or other shady projects.
